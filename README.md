# Models

Occasionally we create 3D-printable models to make our lives easier. We make these freely available (both the source files and STL files) for you to do with as you like.

## Categories

- [19" rack-mountable](rackable/README.md): Models to help us put equipment into 19" racks.
- [SKÅDIS](skadis/README.md): Models which mount to IKEA SKÅDIS pegboards.
- [Radio mic holder](radio-mic-holder/README.md): A stand for our two radio mic belt packs.
